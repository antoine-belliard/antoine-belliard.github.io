Codex Geronimus

Codex Geronimus est un jeu de gestion / stratégie en 3D isométrique dans lequel le joueur doit protéger un peuple de dangereux monstres. Seul problème : il ne parle pas la langue. Ce Game Design Document a été réalisé en adéquation avec les principes du Rational Game Design. Téléchargement du PDF.

  • Date de création : Octobre 2022
  • Créé avec : Moi-même
  • Rôle joué : Game Design, Art.

Nekomimic / Mimic Gotta Sneak

Mimic Gotta Sneak est un jeu de plateforme dans lequel le joueur incarne un mimic qui doit échapper à ses poursuivants en se cachant dans un donjon sombre. Créé dans le cadre de la Code Game Jam 2021, ce jeu a remporté le prix du design, une deuxième fois consécutive. Lien de téléchargement.

  • Date de création : Janvier 2021
  • Temps de création : 30 heures
  • Créé avec : Team MIAOU
  • Rôle joué : Game design, art, animations
  • Contrôles : Flèches directionnelles pour se déplacer. Flèche du bas pour se cacher des ennemis avant d'être repéré.

Epitaph : A matter of life and death

Epitaph est un jeu d'énigme en coop dans lequel la Vie et la Mort doivent utiliser leurs pouvoirs pour arriver à la fin du niveau. Ce jeu a été créé dans le cadre de la Code Game Jam 2020, qui était placée sous le thème de "Cute and Creepy". Epitaph a obtenu le prix du design de la compétition. Lien de téléchargement.

  • Date de création : Janvier 2020
  • Temps de création : 30 heures
  • Créé avec : Team Ubihard
  • Rôle joué : Game design, programmation, pixel-art, animations
  • Contrôles :
    • Joueur 1 : Flèches directionnelles pour se déplacer.
    • Joueur 2 : ZQSD pour se déplacer (clavier azerty).
