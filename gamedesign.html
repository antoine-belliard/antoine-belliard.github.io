Bienvenue sur mon e-portfolio!

Je suis un étudiant en game design vivant à Montpellier depuis 4 ans. J'ai créé ce site pour répertorier mes créations et partager mon CV. Vous trouverez ci-dessous les projets les plus importants sur lesquels j'ai travaillé!

Je recherche un stage de 3 à 6 mois (négociable) en tant que technical game designer junior, gameplay programmer ou assistant producer dans le domaine du jeu vidéo ou du jeu de société, dans la période à partir du 1er avril 2023.

Mes centres d'intérêts sont le jeu vidéo, l'histoire, et le jeu de rôle que je pratique en tant que joueur et MJ depuis 4 ans. Mes genres de jeu vidéo préférés sont les jeux de stratégie, les soulslike, et les jeux narratifs. Les jeux de rôles auxquels je joue sont principalement Vampire the Masquerade et Call of Cthulhu. En ce moment, j'écoute pas mal de Death Grips, d'électro et de folk punk.

Kratos Therapy

Dans Kratos Therapy, Kratos, de retour dans le monde réel après la guerre, est forcé par son fils à mettre le pied dans le bureau d'une psychologue. Trouvera-t-il les bons mots pour échapper à ses rêves et briser le cycle de la violence? Ce jeu vidéo, mélange entre jeu d'action et visual novel, a été réalisé comme un détournement de God Of War (2018). Lien de téléchargement. Téléchargez aussi le GDD!

  • Date de création : Décembre 2022
  • Créé avec : Gabriel, Chloé, Alexandre
  • Rôle joué : Combat Design et programmation (Catch). Analyse de gameplay. Intégration d'assets sur unity.

Zealothan

Détournement de Zenyatta (personnage Overwatch) réalisé sous la forme de Scope Vision pour la Licence Pro Jeu Vidéo en 2022. Téléchargement du PDF.

  • Date de création : Novembre 2022
  • Créé avec : Matt, Rémy, Deborah
  • Rôle joué : Game design des capacités, analyse historique et politique.

Codex Geronimus

Codex Geronimus est un jeu de gestion / stratégie en 3D isométrique dans lequel le joueur doit protéger un peuple de dangereux monstres. Seul problème : il ne parle pas la langue. Ce Game Design Document a été réalisé en adéquation avec les principes du Rational Game Design. Téléchargement du PDF.

  • Date de création : Octobre 2022
  • Créé avec : Moi-même
  • Rôle joué : Game Design, Character Design, Dessins.

Nekomimic / Mimic Gotta Sneak

Mimic Gotta Sneak est un jeu de plateforme dans lequel le joueur incarne un mimic qui doit échapper à ses poursuivants en se cachant dans un donjon sombre. Créé dans le cadre de la Code Game Jam 2021, ce jeu a remporté le prix du design, une deuxième fois consécutive. Lien de téléchargement.

  • Date de création : Janvier 2021
  • Temps de création : 30 heures
  • Créé avec : Team MIAOU
  • Rôle joué : Game Design 3C, Direction Artistique, Character Design, Animations.
  • Contrôles : Flèches directionnelles pour se déplacer. Flèche du bas pour se cacher des ennemis avant d'être repéré.

One Use Hero

One Use Hero est un prototype de jeu créé en une journée à l'occasion d'une mini game jam en solo. Le thème de la compétition était "à usage unique". Dans One Use Hero, le protagoniste doit affronter des vagues d'ennemis en jetant son arme après chaque utilisation. Lien de téléchargement.

  • Date de création : Été 2020
  • Temps de création : 24 heures
  • Créé avec : Moi-même
  • Rôle joué : Game Design, Programmation, Pixel-Art, Animations.

Epitaph : A matter of life and death

Epitaph est un jeu d'énigme en coop dans lequel la Vie et la Mort doivent utiliser leurs pouvoirs pour arriver à la fin du niveau. Ce jeu a été créé dans le cadre de la Code Game Jam 2020, qui était placée sous le thème de "Cute and Creepy". Epitaph a obtenu le prix du design de la compétition. Lien de téléchargement.

  • Date de création : Janvier 2020
  • Temps de création : 30 heures
  • Créé avec : Team Ubihard
  • Rôle joué : Game Design 3C, Programmation capacités d'un personnage, Character Design.
  • Contrôles :
    • Joueur 1 : Flèches directionnelles pour se déplacer.
    • Joueur 2 : ZQSD pour se déplacer (clavier azerty).
